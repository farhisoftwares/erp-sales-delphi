# ERP Sales Delphi

## Project Documentation

This project is an ERP Sales application built using Delphi.

### Features
- User Management
- Sales Tracking
- Inventory Management
- Reporting

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/farhisoftwares/erp-sales-delphi.git
   ```
2. Open the project in Delphi IDE.
3. Build and run the application.

### Usage
- Log in with your credentials to access features.
- Use the menu to navigate through different modules:
  - Sales
  - Inventory
  - Reports
